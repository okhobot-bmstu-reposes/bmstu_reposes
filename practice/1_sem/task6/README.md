# Корзина интернет-магазина (Java)

Консольное приложение для работы с корзиной покупок.

## Запуск

1. Убедитесь, что установлена **Java 17+**.
2. Клонируйте репозиторий:
    git clone <URL-репозитория>

1. Перейдите в папку проекта и скомпилируйте:
   javac *.java
   
2. Запустите:
   java Main
   

# Структура

- Product.java — класс товара
- Cart.java — класс корзины
- Main.java — демонстрация работы
